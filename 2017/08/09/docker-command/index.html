Docker 命令操作手册，方便查询使用。

运行

docker run

–name 指定容器名
-p 指定端口映射

-v 挂载数据卷或者本地目录映射 :ro 挂载为只读
-d 后台持续运行
-i 交互式操作
-t 终端
-rm 容器退出后随之将其删除(与-d 冲突)
