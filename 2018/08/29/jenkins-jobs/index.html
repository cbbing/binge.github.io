Jenkins API

Jenkins提供了一套远端访问的API，目前有3种方式：

  • XML

  • JSON with JSONP support

  • Python

远程API可以执行以下这些功能：

  • 创建/复制 job

  • 触发一个新的构建

  • 查询Jenkins的运行状态
